## 🚀 NLP Projects

This repository also includes hands-on projects to apply NLP concepts:

### 📌 1. Email Spam Classifier
- Classifies emails as **Spam or Not Spam**
- Techniques: TF-IDF, Logistic Regression, EDA
- Focus: Text preprocessing + classification pipeline

---

### 📌 2. Movie Review Sentiment Analysis
- Predicts sentiment (**Positive / Negative**) from reviews
- Techniques: LSTM, BiLSTM, Text Preprocessing
- Dataset: IMDB

---

### 📌 3. Twitter Sentiment Analysis
- Analyzes tweets to detect sentiment
- Handles noisy and short text
- Focus: real-world social media data

---

### 📌 4. Fake News Detection
- Classifies news as **Real or Fake**
- Techniques: TF-IDF + ML / Deep Learning
- Use case: misinformation detection

---

### 📌 5. Duplicate Question Detection
- Identifies whether two questions are similar
- Dataset: Quora Question Pairs
- Focus: semantic similarity
